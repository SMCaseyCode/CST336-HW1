What is MySQL?

MySQL is a database management system. Simply put, MySQL is a structured collection of data which can be organized at mass scale and used in anything from web applications to full production products. MySQL's databases are 'relational' which means data is stored in separate tables instead of in one large object. MySQL's software is also open source! This means anyone can use and modify the software.

Where is MySQL Used?

MySQL is an incredibly valuable and powerful tool. MySQL is used in these applications, many of which you have probably heard of:

Facebook, Twitter, Netflix, Uber, and Shopify
